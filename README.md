# JavaScript Practice & Review

A collection of small JavaScript projects for reviewing concepts and experimenting with different techniques.

## About This Repository

This repository contains various mini-projects created for practice and skill review. Each project focuses on specific JavaScript concepts or techniques.

> **Note:** This repository structure may change in the future. Current projects may be reorganized into separate branches with this becoming the master/main branch.

## Projects

### Infinite Card Carousel
- **Focus:** DOM manipulation, cloning nodes, requestAnimationFrame
- **Features:** Seamless infinite scrolling carousel without animation jerk
- **Technologies:** Vanilla JavaScript, CSS animations

## Purpose

- Review and reinforce JavaScript fundamentals
- Experiment with different implementation approaches
- Build a reference library of reusable patterns
- Practice clean, maintainable code
